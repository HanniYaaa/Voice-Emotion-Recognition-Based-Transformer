# Voice Emotion Recognition

This project focuses on speech emotion recognition using deep learning models.

## Models
- CNN for audio-based emotion classification
- BiLSTM for temporal feature learning
- Transformer for speech emotion recognition

## Features
- MFCC
- Chroma
- RMS
- Spectral Contrast

## Tools
- Python
- TensorFlow / Keras
- Librosa
- FastAPI
- Flutter (Mobile Application)

## Dataset
- Thai Speech Emotion Dataset (THAI-SER)
